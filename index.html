Emacs China ELPA 镜像

ELPA 即 Emacs Lisp Package Archive，是存放 Emacs Lisp 安装包的地方。目前存在多个 ELPA，本镜像支持的 ELPA 有：

ELPA 源地址 镜像地址
GNU ELPA http://elpa.gnu.org/packages/ http://elpa.zilongshanren.com/gnu/
MELPA http://melpa.org/packages/ http://elpa.zilongshanren.com/melpa/
MELPA Stable https://stable.melpa.org/packages/ http://elpa.zilongshanren.com/melpa-stable/
Marmalade http://marmalade-repo.org/packages/ http://elpa.zilongshanren.com/marmalade/
Org http://orgmode.org/elpa/ http://elpa.zilongshanren.com/org/
Sunrise Commander http://joseito.republika.pl/sunrise-commander/ http://elpa.zilongshanren.com/sc/

注意：由于 emacs-china.org 未备案，本镜像用了子龙山人提供的子域名：elpa.zilongshanren.com。万一日后 emacs-china.org 备案通过，elpa.zilongshanren.com 会继续工作或重定向至 elpa.emacs-china.org。
